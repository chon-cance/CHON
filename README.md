## 📚 프로젝트 소개
* **촌스럽게**는 게스트와 호스트를 연결해주는 사이트입니다.
* 촌캉스 숙소들의 정보를 검색할 수 있고 제공합니다.
* 숙소예약 신청시 게스트와 호스트에게 각각 알림톡을 보냅니다.
* 알림톡의 버튼url로 게스트와 호스트가 예약 현황을 확인하고 관리할 수 있습니다.
